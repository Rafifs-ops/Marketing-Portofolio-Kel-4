Marketing Funnel Manage

Saya memetakan dan mengatur setiap langkah dalam marketing funnel secara terstruktur.

Manajemen Proyek dengan ClickUp

Untuk memastikan setiap strategi berjalan dengan lancar, saya mengandalkan ClickUp sebagai platform manajemen proyek utama. Saya mendesain struktur ruang kerja dengan membuat folder untuk setiap fase funnel, membuat daftar tugas untuk setiap PIC, dan mengatur custom fields agar alur kerja tim berjalan lancar.

Menyusun deskripsi tugas lengkap (brief), menetapkan PIC untuk Awareness, Consideration, Purchase, Retention, dan Advocacy, lalu mengatur deadline dan KPI masing‑masing, serta memonitor progress harian melalui dashboard ClickUp untuk memastikan target terpenuhi.
